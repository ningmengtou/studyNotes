##### 移动端基本配置

```
<meta name="viewport" content="width=device-width, initial-scale=1.0" />

<script type="text/javascript">
    var deviceWidth = document.documentElement.clientWidth;
    if(deviceWidth > 750) {
      deviceWidth = 750;
    }
    $("html").css("font-size", deviceWidth / 7.5 + 'px');
  </script>
```
